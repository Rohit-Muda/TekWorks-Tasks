# Student Marks Collection

This is a simple Streamlit application that stores and analyzes student marks using a MySQL database.

---

## Features

In this application, we perform the following steps:

1. Store student details such as:
   - Roll Number  
   - Name  
   - Subject  
   - Marks  

2. Display all student records in a table format.

3. Analyze student performance by:
   - Calculating average marks
   - Calculating pass percentage
   - Finding the top scorer

4. Visualize results using:
   - Bar chart (Subject vs Average Marks)
   - Pie chart (Pass / Fail ratio)
